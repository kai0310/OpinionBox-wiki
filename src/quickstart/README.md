# Quickstart

## Install OpinionBox

It is recommended to use Laravel Sail as the development environment.

```Bash
$ git clone https://github.com/kai0310/OpinionBox.git
$ cd OpinionBox

$ composer install
$ ./vendor/bin/sail up -d
```

Let's get access [http://localhost](http://localhost)

## Requirement

- Git
- PHP(7.3<=)
- Composer
- Node.js
